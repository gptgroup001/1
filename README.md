# Calculator Web App

This repository contains a simple Flask-based calculator. Start the server with:

```bash
python calculator.py
```

Make sure Flask is installed:

```bash
pip install Flask
```

Available endpoints:

- `/add?a=<num>&b=<num>`
- `/sub?a=<num>&b=<num>`
- `/mul?a=<num>&b=<num>`
- `/div?a=<num>&b=<num>`
